# CalendarApp Docs

Public documentation for onboarding new calendar sources into CalendarApp.

## Scope

This repository documents source contribution workflows only:
- schema-first onboarding for external event sources
- HtmlLite schema design and submission patterns
- autonomous AI onboarding runbooks and prompt templates

This repository does not document code contribution for private CalendarApp application repositories.

## Audience Tracks

- Human source contributors: use the fail-safe process and HtmlLite reference docs.
- AI agents: use the autonomous runbook and deterministic output contract.

## Start Points

- Site landing page: `docs/index.md`
- Human onboarding workflow: `docs/source-schemas/fail-safe-process.md`
- AI autonomous workflow: `docs/ai/source-onboarding/runbook.md`
- Chat-only fallback workflow: `docs/ai/source-onboarding/prompt-template.md`
